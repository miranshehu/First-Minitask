What are Lists in html?
HTML lists allow web developers to group a set of related items in lists.

Lists can be ordered and unordered lists.

An example of ordered lists is this which typicaly shows with numbers:

  1. Paragraph 1
  2. Paragraph 2
  3. Paragraph 3

While unordered lists show with bulletpoints and are used when the order is not primary:

  • Paragraph 1
  • Paragraph 2
  • Paragraph 3
